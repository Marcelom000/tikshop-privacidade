Política de Privacidade

Versão 1.0 Última atualização Março de 2026 Aplicável a Extensão Chrome TikShop Analyzer
01

Introdução

A presente Política de Privacidade descreve como o TikShop Analyzer ("extensão", "nós") coleta, usa e protege as informações dos usuários ("você") ao utilizar nossa extensão para o Google Chrome.

Ao instalar e utilizar o TikShop Analyzer, você concorda com os termos descritos nesta política.

02

Dados Coletados

O TikShop Analyzer coleta apenas o mínimo necessário para o funcionamento da extensão:

  • Chave de licença inserida pelo usuário para validação de acesso
  • Identificador único do dispositivo (gerado localmente) para vincular a licença a um único navegador
  • Dados de produtos do TikTok Shop (preço, nome, avaliação, vendas) — processados localmente, não armazenados em nossos servidores
Não coletamos dados pessoais identificáveis

Importante: Os dados de produtos capturados da página do TikTok Shop são processados diretamente no seu navegador e utilizados apenas para exibir os cálculos na extensão. Esses dados não são enviados, armazenados ou compartilhados com terceiros.

03

Como Usamos os Dados

Os dados coletados são utilizados exclusivamente para:

  • Validar sua licença de acesso à extensão junto ao nosso servidor
  • Garantir que cada licença seja utilizada em apenas um dispositivo
  • Calcular margens de lucro e preços mínimos de venda localmente no seu navegador
04

Armazenamento de Dados

A chave de licença e o identificador do dispositivo são armazenados localmente no seu navegador através da API chrome.storage.local, acessível apenas pela própria extensão.

No nosso servidor de validação (N8N + Supabase), armazenamos apenas a chave de licença, o identificador do dispositivo vinculado e o status da licença (ativa/inativa). Não armazenamos dados pessoais como nome completo, CPF, endereço ou dados de pagamento.

05

Compartilhamento de Dados

Não vendemos nem compartilhamos seus dados

Os dados coletados não são vendidos, alugados ou compartilhados com terceiros para fins comerciais ou publicitários. O único compartilhamento ocorre com nosso servidor de validação de licenças, operado exclusivamente por nós.

06

Permissões da Extensão

O TikShop Analyzer solicita as seguintes permissões do Chrome:

  • activeTab — Para acessar o conteúdo da página de produto do TikTok Shop atualmente aberta
  • storage — Para salvar sua chave de licença localmente no navegador
  • tabs — Para identificar se a aba ativa é uma página de produto do TikTok Shop
  • host_permissions (tiktok.com) — Para executar o script de captura de dados nas páginas do TikTok Shop
07

Seus Direitos

Você pode a qualquer momento:

  • Desinstalar a extensão, removendo todos os dados armazenados localmente
  • Solicitar a exclusão da sua chave de licença do nosso servidor
  • Solicitar informações sobre quais dados temos vinculados à sua licença
08

Alterações nesta Política

Podemos atualizar esta Política de Privacidade periodicamente. Alterações significativas serão comunicadas através da atualização da extensão. O uso continuado da extensão após as alterações constitui aceitação da nova política.
